# grafico
Após analisar os dados sobre preços médios de carros, algumas observações e insights se destacaram. No gráfico de barras que mostra o preço médio por tipo de combustível, percebi que os carros movidos a etanol têm preços médios mais altos em comparação aos movidos a gasolina e flex. Isso pode indicar uma maior valorização do etanol no mercado ou uma oferta reduzida desses veículos.

Além disso, o gráfico de linhas que representa a tendência dos preços médios ao longo dos anos de modelo revela um aumento gradual, especialmente nos modelos mais recentes. Isso sugere que a demanda por carros novos está impulsionando os preços, refletindo a valorização de tecnologias e recursos mais avançados.

Por fim, o histograma da distribuição dos preços médios mostra uma concentração significativa de veículos na faixa de preços mais baixos, com uma cauda longa para os preços mais altos. Essa distribuição indica que, embora haja muitos carros acessíveis, existe uma elite de veículos que alcança preços muito superiores, o que pode ser relevante para estratégias de mercado e segmentação de clientes.
